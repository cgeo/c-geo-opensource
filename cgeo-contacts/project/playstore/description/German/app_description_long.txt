Dies ist ein Kontakte-Plugin für c:geo. Das Plugin funktioniert nur, wenn c:geo installiert ist und ermöglicht direkten Zugriff auf das Adressbuch beim Anklicken eines Usernamen in c:geo.

Funktionsweise: 
Wenn ein Username (z.B. im Cache-Logbuch) angeklickt wird, wird im Menü das Öffnen des Kontaktes angeboten. Dies erlaubt es den User direkt anzurufen oder anzuschreiben.

Bemerkung: 
Damit das Plugin den Usernamen zu einem Adressbuch-Eintrag zuordnen kann, muss der Username im Namens- oder Spitznamen-Feld eingetragen sein.
