Ceci est un plugin de contacts pour c:geo, une application de géocaching open source.

Utilisation :
Lorsque vous cliquez sur un nom d'utilisateur dans c:geo (par ex. dans le journal de la cache) le menu contiendra un élément pour ouvrir le contact correspondant dans votre carnet d'adresses. Cela vous permettra directement d'appeler ou d'écrire un message à l'utilisateur.

Remarque:
Afin que le plugin puisse associer l'utilisateur géocaching et votre contact, l'utilisateur géocaching doit être présent dans le champ nom ou alias de votre contact.
